🚀 Démarrage
Comment rejoindre une salle ?▼
Sur la page d'accueil, entre le code de ta salle (ex: 406 ou 4eme-406) dans le champ "Code de la salle" et clique sur Rejoindre. Tu peux aussi parcourir les classes du collège et cliquer directement sur ta salle.
Faut-il créer un compte ?▼
Non ! ClassChat fonctionne sans inscription. Tu entres juste ton pseudo, tu choisis ta salle, et c'est parti. Aucun email, aucun mot de passe requis pour le chat.
Comment créer ma propre salle ?▼
Clique sur le bouton + Créer dans le menu principal. Donne un nom à ta salle (ex: ton numéro de classe), choisis si elle est publique ou privée, et partage le code avec tes amis.
Comment changer mon pseudo ou ma photo ?▼
Clique sur ton avatar en haut ou dans le menu ⚙️ Réglages. Tu peux changer ton pseudo, uploader une photo, créer un avatar avec l'outil intégré, ou même transformer ta photo en avatar.
💬 Fonctionnalités
Comment envoyer un message vocal ?▼
Maintiens le bouton 🎤 microphone appuyé pour enregistrer. Relâche pour envoyer. Si tu glisses ton doigt vers la gauche avant de relâcher, le message est annulé.
Comment envoyer une photo ou une vidéo ?▼
Clique sur l'icône 📎 trombone dans la barre de saisie. Tu peux choisir un fichier depuis ta galerie. Tu peux aussi prendre une photo directement avec la caméra depuis le bouton 📷 dans le menu.
Comment jouer au Puissance 4 ?▼
Va dans tes Messages privés (DM) avec un ami, puis clique sur le bouton 🎮 en haut de la conversation. Le jeu se lance directement dans le chat !
Comment lancer un appel audio ou vidéo ?▼
Dans une salle, clique sur l'onglet 📞 Appels dans la barre du bas. Tu peux lancer un appel audio ou vidéo. Tous les membres de la salle peuvent rejoindre.
C'est quoi ClassChat Studio ?▼
Studio est la plateforme vidéo de ClassChat — comme un YouTube pour ta classe. Tu peux publier des vidéos, les commenter, les liker et les partager. Accès via classchat.vercel.app/studio.
🎬 ClassChat Studio
Comment publier une vidéo sur Studio ?▼
Va sur Studio et connecte-toi (ou crée un compte). Clique sur ＋ Publier dans la navigation, glisse ou sélectionne ta vidéo, ajoute un titre, et clique Publier. C'est tout !
Puis-je éditer ou rogner mes vidéos ?▼
Oui ! Dans Studio, va dans l'onglet ✂️ Montage et sélectionne une de tes vidéos. Tu peux définir un point de début/fin, appliquer un filtre couleur et ajouter un texte en surimpression avant de republier.
Comment supprimer une vidéo ?▼
Ouvre la vidéo en question sur Studio. Le bouton 🗑️ Supprimer apparaît en bas si tu es le créateur. Confirme la suppression — l'action est irréversible.
Qui peut voir mes vidéos ?▼
Toutes les vidéos publiées sur Studio sont visibles par tous les utilisateurs connectés. Il n'y a pas encore de vidéos privées — pense à ne pas publier de contenu trop personnel.
⚙️ Technique & Problèmes
L'app ne charge pas ou est lente▼
Essaie de rafraîchir la page (F5 ou tirer vers le bas sur mobile). Vérifie ta connexion internet. Si le problème persiste, vide le cache de ton navigateur (Ctrl+Shift+Delete).
Je ne vois pas les nouveaux messages▼
ClassChat utilise une connexion temps réel. Si les messages ne s'actualisent pas, vérifie ta connexion et recharge la page. Il peut aussi y avoir un délai si tu es sur un réseau instable (ex: 4G faible).
Comment installer ClassChat sur mon téléphone ?▼
Ouvre ClassChat dans Chrome sur Android ou Safari sur iPhone. Un bouton "Installer" ou "Ajouter à l'écran d'accueil" apparaît. L'app s'installe comme une vraie application, sans passer par le Play Store !
Mes données sont-elles sécurisées ?▼
ClassChat ne collecte aucune donnée personnelle. Les messages sont stockés de façon sécurisée sur Supabase. Les salles privées sont protégées par code. Aucun email ni mot de passe n'est requis pour le chat.
ClassChat fonctionne sans connexion ?▼
Non, ClassChat nécessite une connexion internet pour fonctionner (chat temps réel). Mais une fois installée comme app, la page de connexion se charge très vite même sur 4G. Pour une utilisation hors-ligne, il n'y a pas de mode offline pour l'instant.
